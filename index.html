SDK Reference Manual

The Avalon SDK Reference Manual is generated with Doxygen. To generate the Reference Manual, type the following:

cd $TCF_HOME/docs # this directory
sudo apt-get update
sudo apt-get install -y make doxygen texlive-full graphviz
make

Documentation generated will be here:

  • $TCF_HOME/docs/refman/html/HTML documentation
  • $TCF_HOME/docs/refman/man/man3man page documentation
  • $TCF_HOME/docs/refman/latexLaTeX documentation
  • $TCF_HOME/docs/refman/refman.pdfPDF documentation (generated from LaTex)
